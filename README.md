# 📊 Startup Financial KPI Analysis

## 📌 Project Overview
This project analyzes the financial performance of a startup using Excel, Python, and Tableau. It focuses on measuring key financial KPIs, identifying business trends, and creating interactive dashboards to support data-driven decision-making.

## 🎯 Objectives
- Analyze startup financial performance.
- Track Revenue, Expenses, Profit, CAC, LTV, and Burn Rate.
- Visualize monthly business trends.
- Build interactive dashboards for better business insights.

## 🛠️ Tools & Technologies
- Microsoft Excel
- Python
- Pandas
- Matplotlib
- Tableau

## 📂 Dataset
The project uses a startup financial dataset containing monthly business information such as:
- Revenue
- Expenses
- Marketing Cost
- New Customers
- Active Customers
- Subscription Revenue
- Profit
- Profit Margin
- CAC
- LTV
- Burn Rate

## 📈 Key Performance Indicators (KPIs)
- Total Revenue
- Total Expenses
- Net Profit
- Profit Margin
- Customer Acquisition Cost (CAC)
- Lifetime Value (LTV)
- LTV:CAC Ratio
- Burn Rate
- Active Customers

## 📊 Visualizations
- Monthly Revenue Trend
- Monthly Expenses Trend
- Monthly Profit Trend
- Active Customers Growth
- LTV vs CAC Comparison
- Interactive Tableau Dashboard
- Excel KPI Dashboard

## 📌 Project Workflow
1. Data Collection
2. Data Cleaning
3. KPI Calculation
4. Exploratory Data Analysis (EDA)
5. Data Visualization
6. Dashboard Development
7. Business Insights

## 💡 Key Insights
- Revenue showed consistent monthly growth.
- Profit increased steadily throughout the year.
- Active customers increased over time.
- LTV remained higher than CAC, indicating efficient customer acquisition.
- Business performance improved month by month.

## 🚀 Project Outcome
Developed an end-to-end financial analytics solution using Excel, Python, and Tableau to monitor startup performance and support business decision-making through interactive dashboards and visual analytics.
